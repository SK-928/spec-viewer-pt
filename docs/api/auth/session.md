# セッション管理

リフレッシュトークンを軸としたセッション寿命と、ローテーション戦略を定義する。

## セッション寿命

| 区分 | 期間 |
|---|---|
| アクティブセッション | 30 分（操作で延長） |
| 最大寿命 | 24 時間 |

## ローテーション

リフレッシュトークンは使用のたびに新規発行し、旧トークンを無効化する（ローテーション）。

- 旧トークンの再利用を検出した場合、当該セッションを即座に無効化する。
- 失効した旧トークンは監査ログに記録する。
