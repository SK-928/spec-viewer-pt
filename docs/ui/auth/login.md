# ログイン画面 仕様書

既存ユーザーが認証を行う画面。

## 画面要素

| 要素 | 型 | 必須 | 備考 |
|---|---|---|---|
| メールアドレス | input | 必須 | |
| パスワード | input | 必須 | マスク表示・表示切替あり |
| ログインボタン | button | — | submit |
| ソーシャルログイン | button | — | Google / GitHub |
| パスワードリセット | link | — | リセット画面へ |

## バリデーション

- メールアドレス: RFC 5322 準拠
- パスワード: 8 文字以上

## 2FA

2要素認証を有効化しているユーザーには、パスワード確認後にコード入力画面へ遷移する。
